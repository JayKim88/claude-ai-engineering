# 장기 자산 증식 워크플로우

**적합 대상**: 기초 재무 상태가 안정된 분 (비상금 확보, 부채 관리 완료)
**목표**: 1-3년 내 자산 증식 기반 구축, 복수 소득원 확보
**전제 조건**: 비상금 3개월분 이상 보유, 고금리 부채 없음

---

## Phase 1: 기초 다지기 (1-3개월)

### 1개월차: 재무 기반 점검
- [ ] 비상금 확인: 월 생활비 × 6개월 = ___만원 (목표)
  - 부족하면 Phase 1 동안 비상금 확보 우선
- [ ] 보험 정리:
  - 실손보험 1개 (필수)
  - 종신보험 → 정기보험 전환 검토 (보험료 절약)
  - 중복 가입 해지
  - 목표: 월 보험료 수입의 5% 이내
- [ ] 고정비 최적화:
  - 통신비: 알뜰폰 전환 검토
  - 구독: 미사용 서비스 해지
  - 카드: 혜택 최적화 (주 사용처 기준)
- [ ] 목적별 통장 시스템 구축:
  ```
  급여통장 → 자동이체
  ├─ 비상금 통장 (파킹통장)
  ├─ 투자 통장 (CMA)
  ├─ 생활비 통장 (체크카드 연결)
  └─ 목표 통장 (여행/큰 지출)
  ```

### 2개월차: 절세 계좌 완성
- [ ] ISA 계좌 활성화 (비과세 혜택)
  - 월 자동이체 설정: ___만원
  - ETF 포트폴리오 구성
- [ ] 연금저축 활성화 (세액공제)
  - 월 50만원 (연 600만원 한도)
  - TDF 또는 글로벌 ETF 선택
- [ ] IRP 추가 검토 (연금저축 + IRP 합산 연 900만원 공제)
- [ ] 주택청약저축 확인 (미가입 시 가입, 월 2-10만원)

### 3개월차: 자동 투자 시스템 완성
- [ ] 급여일 기준 자동 분배 최종 세팅:
  ```
  급여일 자동이체
  ├─ 비상금: 10% (목표 달성까지)
  ├─ ISA: 15%
  ├─ 연금저축: 15%
  ├─ 생활비: 50%
  └─ 자유: 10%
  ```
- [ ] 모든 자동이체 정상 작동 확인
- [ ] 3개월간의 지출 패턴 분석 → 비율 미세 조정

**Phase 1 체크포인트**: 비상금 6개월분, 절세 계좌 3종 활성, 자동이체 정상 가동

---

## Phase 2: 투자 본격화 (4-6개월)

### 4개월차: 포트폴리오 확장
- [ ] 현재 포트폴리오 리뷰
- [ ] 자산 배분 결정 (나이/리스크에 따라):
  ```
  보수적 (저위험): 주식 ETF 40% + 채권 ETF 40% + 현금 20%
  균형형 (중위험): 주식 ETF 60% + 채권 ETF 25% + 대안투자 15%
  적극적 (고위험): 주식 ETF 80% + 대안투자 15% + 현금 5%
  ```
- [ ] 글로벌 분산:
  - 한국 (KODEX 200): 20-30%
  - 미국 (TIGER S&P500): 40-50%
  - 기타 선진국/신흥국: 10-20%
- [ ] 섹터 ETF 검토 (관심 분야: IT, 헬스케어, 클린에너지 등)

### 5개월차: 투자 지식 심화
- [ ] 투자 도서 2권 읽기
  - 기본: "현명한 투자자" (그레이엄) 또는 "이기는 투자" (린치)
  - 심화: "원칙" (달리오) 또는 "Poor Charlie's Almanack" (멍거)
- [ ] 재무제표 기초 학습:
  - 매출, 영업이익, 순이익
  - PER, PBR, ROE 의미
  - 현금흐름표 읽기
- [ ] 관심 기업 3개 분석 연습

### 6개월차: 배당 전략 검토
- [ ] 배당주/배당 ETF 학습:
  - ARIRANG 고배당주 (한국)
  - SCHD, VYM (미국 배당 ETF)
- [ ] 배당 재투자(DRIP) 설정
- [ ] 6개월 포트폴리오 성과 리뷰
- [ ] 리밸런싱 실행 (목표 비율에서 5%+ 이탈 시)

**Phase 2 체크포인트**: 글로벌 분산 포트폴리오 구축, 재무제표 기초 이해

---

## Phase 3: 소득 다각화 (7-12개월)

### 7-8개월차: 부업 1개 런칭
- [ ] side-hustle-launch.md 워크플로우 시작
- [ ] 본업 + 투자 + 부업 = 3개 소득원 확보
- [ ] 부업 수입 100% 투자로 배분

### 9-10개월차: 소득 최적화
- [ ] 본업 연봉 협상 / 스킬업 검토
  - 연봉 5% 인상 = 연간 투자금 ___만원 증가
- [ ] 부업 수입 안정화 (월 목표 달성 여부)
- [ ] 불필요 지출 2차 점검

### 11-12개월차: 1년 리뷰 & 확장
- [ ] 1년 재무 성과 정리:
  ```
  총 자산 변화: ___만원 → ___만원 (증가: ___만원)
  투자 수익률: ___%
  저축률: ___%
  소득원 수: ___개
  ```
- [ ] 2년차 목표 수립:
  - 투자 금액 증액
  - 새로운 자산 클래스 검토 (부동산, 대안투자)
  - 부업 확장 또는 2번째 부업

**Phase 3 체크포인트**: 3개 이상 소득원, 월 투자금 수입의 30% 이상

---

## Phase 4: 자산 증식 가속 (1-3년)

### 1-2년차: 포트폴리오 확대
- [ ] 투자 금액 점진적 증액 (수입 증가분의 50% 이상)
- [ ] 개별주 투자 시작 (ETF 비중 70% 유지하며)
  - 관심 기업 워치리스트 10개 관리
  - 분기 실적 체크 루틴
- [ ] 해외 직접투자 검토 (미국 개별주)

### 2-3년차: 부동산 & 대안투자 검토
- [ ] 부동산 투자 학습:
  - 전세 레버리지, 소형 아파트/오피스텔
  - 리츠(REITs) ETF 검토 (부동산 간접 투자)
- [ ] 대안투자 검토:
  - 금 ETF (인플레이션 헤지)
  - P2P 투자 (소액, 분산)
- [ ] 법인 설립 검토 (부업 매출 증가 시 절세 효과)

**Phase 4 체크포인트**: 순자산 1억+ 달성, 4개 이상 소득원

## 마일스톤 요약

| 시점 | 목표 | 핵심 행동 |
|------|------|----------|
| 3개월 | 재무 기반 완성 | 비상금 + 절세계좌 + 자동이체 |
| 6개월 | 투자 본격화 | 글로벌 포트폴리오 + 재무지식 |
| 12개월 | 소득 다각화 | 3개 소득원 + 투자 30%+ |
| 2년 | 자산 증식 가속 | 개별주 + 부동산 검토 |
| 3년 | 재무 자유 기반 | 순자산 1억+, 패시브 인컴 시작 |

## 핵심 원칙

1. **자동화가 핵심**: 의지력에 의존하지 않는 시스템 구축
2. **소득 증대 > 절약**: 절약의 한계는 있지만 소득의 한계는 없다
3. **투자는 일찍, 길게**: 복리의 시간을 최대화
4. **분산하되 집중**: 너무 많은 것에 분산하지 않기
5. **매월 리뷰**: 월 1회 30분 재무 점검은 필수
