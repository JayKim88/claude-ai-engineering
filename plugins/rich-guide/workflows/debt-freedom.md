# 빚 탈출 워크플로우

**적합 대상**: 대출/부채가 있어 재테크 시작이 어려운 분
**목표**: 고금리 부채 청산 → 투자 시작 기반 마련
**총 소요 기간**: 상황에 따라 3-24개월

---

## Phase 1: 부채 현황 파악 (1주)

### 전체 부채 목록 작성
- [ ] 모든 대출/부채를 한 곳에 정리:

| 구분 | 잔액 | 금리 | 월 상환액 | 남은 기간 |
|------|------|------|----------|----------|
| 카드론 | 만원 | % | 만원 | 개월 |
| 학자금 | 만원 | % | 만원 | 개월 |
| 전세대출 | 만원 | % | 만원 | 개월 |
| 신용대출 | 만원 | % | 만원 | 개월 |
| 기타 | 만원 | % | 만원 | 개월 |

- [ ] 총 부채 합계 계산
- [ ] 월 총 상환액 계산
- [ ] 부채비율 계산: (월 상환액 / 월 수입) × 100
  - 30% 이하: 관리 가능
  - 30-50%: 주의 필요
  - 50% 이상: 긴급 조치 필요

### 부채 분류
- [ ] **긴급 (금리 15%+)**: 카드론, 현금서비스, 대부업 → 즉시 상환 최우선
- [ ] **주의 (금리 7-15%)**: 신용대출, 마이너스 통장 → 대환 검토
- [ ] **관리 (금리 7% 이하)**: 학자금, 전세대출, 주택담보 → 정상 상환 유지

**Phase 1 체크포인트**: 전체 부채 목록 완성, 부채비율 확인

---

## Phase 2: 전략 선택 (2주)

### 상환 전략 결정

**눈사람 방법 (Avalanche)** — 수학적 최적
```
1. 모든 부채의 최소 상환액 유지
2. 여유 자금을 금리가 가장 높은 부채에 집중
3. 최고 금리 부채 상환 완료 → 다음 높은 금리로 이동
4. 반복

장점: 총 이자 지출 최소화
단점: 심리적으로 성취감이 늦게 옴
```

**눈덩이 방법 (Snowball)** — 심리적 최적
```
1. 모든 부채의 최소 상환액 유지
2. 여유 자금을 잔액이 가장 작은 부채에 집중
3. 최소 잔액 부채 상환 완료 → 다음 작은 부채로 이동
4. 반복

장점: 빠른 성취감, 동기부여 유지
단점: 총 이자가 약간 더 많을 수 있음
```

**추천 하이브리드 전략**:
```
금리 15%+ 부채: 눈사람(금리 순) — 이자 폭탄 방지
금리 15% 미만: 눈덩이(잔액 순) — 동기부여 유지
```

### 대환대출 검토
- [ ] 현재 대출 금리와 시장 금리 비교
- [ ] 금리 차이 2%p 이상이면 대환 검토 가치 있음
- [ ] 대환 비용 확인 (중도상환수수료, 인지세, 근저당 설정비)
- [ ] 서민금융진흥원 대환대출 프로그램 확인: https://www.kinfa.or.kr/

### 비상금 확보
- [ ] 부채 상환 중에도 최소 100만원 비상금 유지
- [ ] 비상금 없으면 추가 대출 발생 → 악순환

**Phase 2 체크포인트**: 상환 전략 결정, 대환 가능 여부 확인

---

## Phase 3: 실행 (3주~)

### 자동화 설정
- [ ] 각 부채별 자동 상환 설정 (최소 상환액)
- [ ] 여유 자금 자동이체 → 최우선 상환 대상 추가 납입
- [ ] 급여일 기준 상환 스케줄 작성:
  ```
  급여일 → 자동 분배
  ├─ 50%: 생활비 (최대한 절약)
  ├─ 35%: 부채 상환 (최소상환 + 추가상환)
  ├─ 10%: 비상금 (100만원 확보까지)
  └─ 5%: 자유 사용
  ```

### 지출 절감
- [ ] 3개월치 지출 내역 분석 (카드 명세서)
- [ ] 고정비 점검: 구독 서비스, 보험, 통신비
  - 불필요 구독 해지
  - 보험 중복 가입 확인
  - 알뜰폰 전환 검토 (월 2-3만원 절약)
- [ ] 변동비 줄이기: 외식 → 자취, 카페 → 텀블러
- [ ] 목표: 월 20-50만원 추가 확보 → 전액 상환에 투입

### 추가 소득 투입
- [ ] 부업 검토 (side-hustle-launch.md 워크플로우 참조)
- [ ] 부업 수입 100% 부채 상환에 투입
- [ ] 불필요 물건 판매 (당근마켓, 번개장터)

**Phase 3 체크포인트**: 자동 상환 실행 중, 월 추가 상환 N만원

---

## Phase 4: 부채 청산 후 전환 (부채 0원 달성 시)

### 투자로 전환
- [ ] 부채 상환에 쓰던 금액 → 그대로 투자로 전환
  - 예: 월 30만원 상환 → 월 30만원 ISA 적립
- [ ] first-investment.md 워크플로우로 이동
- [ ] 절대 다시 빚지지 않겠다는 원칙 수립

### 금융 습관 재정립
- [ ] 신용카드 → 체크카드 전환 (과소비 방지)
- [ ] 비상금 목표 상향: 100만원 → 월급 3개월분
- [ ] 매월 재무 점검 루틴 수립

**Phase 4 체크포인트**: 부채 0원 달성, 투자 자동이체 설정 완료
