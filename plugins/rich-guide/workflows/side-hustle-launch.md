# 부업 시작 워크플로우

**적합 대상**: 본업 외 추가 소득원을 만들고 싶은 분
**목표**: 12주 내 월 30-100만원 추가 수입 발생
**총 소요 기간**: 12주 (주 5-10시간 투자 기준)

---

## Phase 1: 강점 발견 & 방향 설정 (1-2주)

### 1주차: 자기 분석
- [ ] 나의 기술/경험 목록 작성:
  ```
  전문 기술: _________________ (예: 프로그래밍, 디자인, 글쓰기, 회계)
  경험/지식: _________________ (예: 마케팅 5년, 영어 능통, 요리)
  관심사:    _________________ (예: 사진, 운동, 반려동물)
  자격증:    _________________ (예: TOEIC 900, 한식조리사, 정보처리기사)
  ```
- [ ] 투자 가능 시간 파악:
  - 평일: 하루 ___시간 (퇴근 후)
  - 주말: 하루 ___시간
  - 주당 총: ___시간
- [ ] 투자 가능 자본: ___만원
- [ ] 수입 목표: 월 ___만원

### 2주차: 부업 유형 선택
- [ ] 3가지 유형 중 선택:

| 유형 | 특징 | 수입 시작 | 예시 |
|------|------|----------|------|
| **즉시 수입형** | 시간 = 돈, 빠른 수입 | 1주 | 배달, 대리운전, 과외 |
| **성장형** | 초기 투자 후 점진적 증가 | 1-3개월 | 프리랜싱, 스마트스토어, 콘텐츠 |
| **자산형** | 장기 투자, 패시브 인컴 | 6개월+ | 전자책, 온라인 강의, 앱 |

- [ ] 선택한 유형: _________________
- [ ] 구체적 부업: _________________

**Phase 1 체크포인트**: 부업 방향 결정, 주당 투자 시간 확정

---

## Phase 2: 시장 조사 & 준비 (3-4주)

### 3주차: 플랫폼 탐색
- [ ] 선택한 부업에 맞는 플랫폼 3개 가입:

| 카테고리 | 플랫폼 |
|----------|--------|
| 프리랜싱 | 크몽, 위시켓, 숨고 |
| 온라인 판매 | 스마트스토어, 쿠팡 마켓플레이스 |
| 콘텐츠 | 유튜브, 네이버 블로그, 브런치 |
| 재능 판매 | 탈잉, 클래스101, 숨고 |
| 배달/서비스 | 쿠팡이츠, 배민커넥트, 카카오 T |
| 중고/리셀 | 당근마켓, 번개장터, 크림 |

- [ ] 각 플랫폼의 수수료/정책 파악
- [ ] 상위 판매자/인기 서비스 분석 (10개)
  - 가격대, 리뷰 수, 서비스 구성 파악

### 4주차: 경쟁자 분석 & 차별화
- [ ] 상위 경쟁자 5명 분석:
  - 강점/약점, 가격, 리뷰 내용
- [ ] 나의 차별화 포인트 정의:
  - 더 싸게? 더 빠르게? 더 전문적으로? 틈새시장?
- [ ] 서비스/상품 기획:
  - 이름: _________________
  - 가격: ___만원
  - 핵심 가치: _________________
- [ ] 필요한 도구/장비 준비

**Phase 2 체크포인트**: 플랫폼 가입 완료, 서비스/상품 기획 완료

---

## Phase 3: MVP 런칭 (5-8주)

### 5주차: 최소 상품/서비스 등록
- [ ] 프로필 완성 (사진, 소개, 포트폴리오)
- [ ] 첫 상품/서비스 등록
  - 가격: 시장가 대비 20-30% 할인 (초기 고객 확보용)
  - 설명: 상세하게, 기대 결과물 명시
  - 이미지: 최소 3장 (전문적으로)
- [ ] 키워드/해시태그 최적화

### 6주차: 첫 고객 확보
- [ ] 지인에게 홍보 (카톡, SNS)
- [ ] 무료 또는 대폭 할인 제공 (리뷰 조건)
  - 목표: 최소 3개의 긍정적 리뷰 확보
- [ ] 첫 고객 경험을 완벽하게 만들기
  - 빠른 응답 (1시간 이내)
  - 기대 이상의 결과물
  - 정중한 리뷰 요청

### 7주차: 피드백 반영
- [ ] 고객 피드백 수집 & 분석
- [ ] 서비스 개선점 반영
- [ ] 가격 재조정 (리뷰가 쌓이면 점진적 인상)
- [ ] 추가 상품/서비스 기획

### 8주차: 마케팅 강화
- [ ] SNS 홍보 (인스타그램, 블로그)
- [ ] 포트폴리오 업데이트 (실제 결과물)
- [ ] 플랫폼 내 광고 검토 (ROI 계산 후)
- [ ] 재구매/소개 할인 정책 수립

**Phase 3 체크포인트**: 유료 고객 5명 이상, 리뷰 3개 이상

---

## Phase 4: 최적화 & 확장 (9-12주)

### 9주차: 수익 분석
- [ ] 시간당 수익 계산:
  ```
  총 수입 / 총 투입 시간 = 시간당 ___원
  목표: 최소 시급 2만원 이상
  ```
- [ ] 비용 정리 (플랫폼 수수료, 도구비, 재료비)
- [ ] 순수익 계산

### 10주차: 프로세스 최적화
- [ ] 반복 업무 템플릿화/자동화
- [ ] 자주 묻는 질문 FAQ 정리
- [ ] 작업 시간 단축 방법 찾기
- [ ] 불필요한 저수익 업무 제거

### 11주차: 확장 전략
- [ ] 추가 플랫폼 진출 (1-2개)
- [ ] 가격 인상 (리뷰/실적 기반)
- [ ] 상위 서비스 패키지 추가
- [ ] 단골 고객 관리 시스템 구축

### 12주차: 3개월 리뷰 & 다음 계획
- [ ] 3개월 성과 정리:
  ```
  총 수입: ___만원
  월 평균: ___만원
  고객 수: ___명
  평균 평점: ___점
  ```
- [ ] 다음 3개월 목표 수립
- [ ] 수입의 투자 배분 계획 (부업 수입 → 투자로)

**Phase 4 체크포인트**: 월 안정 수입 발생, 시간당 2만원+ 달성

## 부업 유형별 빠른 시작 가이드

### 프리랜서 (개발/디자인/글쓰기)
1. 크몽 가입 → 프로필 작성 → 서비스 등록
2. 포트폴리오 3개 준비 (무료 작업도 OK)
3. 초기 가격: 시장가 50% (리뷰 확보 목적)
4. 리뷰 5개 후 정상가로 인상

### 스마트스토어
1. 네이버 스마트스토어 입점 (사업자등록 필요)
2. 위탁판매로 시작 (재고 부담 없음)
3. 상품 10개 등록, 키워드 최적화
4. 네이버 쇼핑 광고 소액 테스트 (일 5,000원)

### 콘텐츠 (블로그/유튜브)
1. 주제 선정 (전문성 + 검색량)
2. 주 2-3회 콘텐츠 발행
3. 3개월(30개 콘텐츠) 후 수익화 검토
4. 애드센스 → 협찬 → 자체 상품 순서

## 세금 & 법률 체크리스트

- [ ] **겸업 금지 확인**: 본업 근로계약서에 겸업 금지 조항 있는지 확인
- [ ] **사업자등록**: 연 매출 4,800만원 이상 예상 시 간이사업자 등록 권장
- [ ] **종합소득세**: 연 부업 소득 300만원+ → 5월 종합소득세 신고 필수
- [ ] **4대 보험**: 프리랜서 소득 → 건강보험료 추가 부과 가능
- [ ] **경비 처리**: 사업 관련 지출 영수증 보관 (세금 절감)
