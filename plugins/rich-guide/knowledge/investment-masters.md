## 1. 벤저민 그레이엄 (Benjamin Graham) - 가치투자의 아버지

**적합 프로필**:
| 항목 | 내용 |
|------|------|
| 레벨 | 입문~중급 |
| 리스크 성향 | 저위험 |
| 최소 자본 | 500만원 (분산 투자 기준) |
| 추천 대상 | 예금이 주 자산인 분, 원금 손실에 민감한 분, 체계적 기준이 필요한 분 |

**대표 저서**: <<현명한 투자자>> (The Intelligent Investor, 1949), <<증권 분석>> (Security Analysis, 1934)

### 핵심 철학
> "투자의 본질은 철저한 분석에 기반하여 원금의 안전과 적절한 수익을 약속하는 행위다. 이 조건을 충족하지 못하면 투기다."

### 투자 전략

#### 전략 1: 안전마진 (Margin of Safety)

```
공식:
  안전마진 = (내재가치 - 현재 주가) / 내재가치 x 100

원칙:
  - 안전마진 30% 이상일 때만 매수
  - 내재가치 10,000원인 주식 → 7,000원 이하에서 매수
  - 안전마진이 클수록 하방 리스크 축소

한국 시장 적용:
  코스피 평균 PBR 약 0.9~1.1배 (2024~2025 기준)
  → PBR 0.5 이하 종목은 그레이엄식 안전마진 후보
  예시: 금융주, 건설주, 지주회사 등 저PBR 업종
```

#### 전략 2: Net-Net 전략 (순유동자산 가치 투자)

```
공식:
  NCAV (순유동자산가치) = 유동자산 - 총부채
  Net-Net 조건: 시가총액 < NCAV x 2/3

해석:
  - 회사를 당장 청산해도 주가보다 많은 돈이 남는 상태
  - 극단적 저평가 종목 발굴법

한국 시장 적용:
  코스피/코스닥에서 Net-Net 종목은 연간 20~40개 수준
  주로 소형주에서 발견됨 (시가총액 500억 이하)

  스크리닝 조건:
  - 유동자산 > 총부채 x 1.5
  - 시가총액 < 순유동자산 x 0.67
  - 최근 3년 영업이익 흑자
```

#### 전략 3: 방어적 투자자 vs 공격적 투자자 분류

```
방어적 투자자 (Defensive Investor):
  - 시간/노력을 최소화하려는 투자자
  - 포트폴리오: 주식 25~75%, 채권 25~75%
  - 종목 기준:
    PER < 15
    PBR < 1.5
    PER x PBR < 22.5
    부채비율 < 100%
    10년 중 7년 이상 흑자
    최소 20개 종목 분산

공격적 투자자 (Enterprising Investor):
  - 분석에 시간과 노력을 투자할 의지가 있는 투자자
  - Net-Net, 회생주, 특수 상황 투자 가능
  - 종목 수: 10~30개
  - 추가 조건: FCF 양수, 배당 이력 5년+

한국 시장에서의 구분:
  방어적 → KODEX 200 + 국고채 ETF 조합
  공격적 → 개별 종목 가치투자 (코스피 저PBR 스크리닝)
```

### 실행 체크리스트

**정량 기준 (반드시 확인)**:
- [ ] PER < 15 (최근 3년 평균 기준)
- [ ] PBR < 1.5 (순자산가치 대비)
- [ ] PER x PBR < 22.5 (그레이엄 기준수)
- [ ] 부채비율 < 100% (총부채 / 자기자본)
- [ ] 유동비율 > 200% (유동자산 / 유동부채)
- [ ] 최근 10년 중 7년 이상 순이익 흑자
- [ ] 배당 지급 이력 5년 이상

**정성 기준 (추가 검토)**:
- [ ] 사업 모델을 한 문장으로 설명할 수 있는가?
- [ ] 10년 후에도 이 회사가 존재할 것인가?
- [ ] 경영진이 주주 이익을 중시하는가?
- [ ] 업종 내 경쟁 지위가 안정적인가?

### 초보자를 위한 시작 가이드

**Step 1**: <<현명한 투자자>> 8장(시장 변동성), 20장(안전마진) 먼저 읽기
**Step 2**: 증권사 HTS/MTS에서 PER, PBR 필터 스크리닝 실습
**Step 3**: 코스피200 구성 종목 중 그레이엄 기준 통과 종목 5개 리스트업
**Step 4**: 각 종목의 최근 5년 재무제표 분석 (영업이익, 부채비율, 배당)
**Step 5**: 가장 확신 있는 2~3개 종목에 소액(50~100만원) 분할 매수

### 대표 투자 성과

- **그레이엄-뉴먼 펀드 (1936~1956)**: 연평균 수익률 약 20% (동기간 S&P 500 대비 약 2배)
- **GEICO 투자**: 1948년 71.25만 달러 투자 → 이후 수십억 달러 가치로 성장
- **Net-Net 전략 백테스트**: 1970~2010년 미국 시장에서 연평균 약 25% 수익률 (Tobias Carlisle 연구)
- **한국 저PBR 전략**: 2003~2023년 코스피 PBR 하위 20% 종목군, 연평균 약 15% 수익 (시장 대비 초과수익)

## 2. 워렌 버핏 (Warren Buffett) - 오마하의 현인

**적합 프로필**:
| 항목 | 내용 |
|------|------|
| 레벨 | 중급 |
| 리스크 성향 | 중위험 |
| 최소 자본 | 1,000만원 (집중 투자 특성상) |
| 추천 대상 | 5년 이상 장기 투자 가능한 분, 기업 분석에 관심 있는 분, 복리 효과를 이해하는 분 |

**대표 저서**: <<버핏의 주주 서한>> (Berkshire Hathaway Annual Letters), <<워렌 버핏 바이블>>

### 핵심 철학
> "훌륭한 회사를 적정한 가격에 사는 것이, 적당한 회사를 훌륭한 가격에 사는 것보다 훨씬 낫다."

### 투자 전략

#### 전략 1: 경제적 해자 (Economic Moat) 분석

```
해자 유형과 한국 시장 예시:

1. 브랜드 파워 (Brand)
   - 글로벌: 코카콜라, 애플
   - 한국: 삼성전자 (반도체 브랜드), 아모레퍼시픽 (화장품)
   → 지표: 매출총이익률 > 40%, 광고비 대비 매출 성장

2. 네트워크 효과 (Network Effect)
   - 글로벌: Visa, Meta
   - 한국: 카카오 (메신저), 네이버 (검색+커머스)
   → 지표: MAU 증가율, 플랫폼 내 거래액 성장률

3. 전환 비용 (Switching Cost)
   - 글로벌: Microsoft, SAP
   - 한국: 더존비즈온 (ERP), 한글과컴퓨터
   → 지표: 고객 이탈률 < 5%, 반복 매출 비중 > 70%

4. 비용 우위 (Cost Advantage)
   - 글로벌: Costco, TSMC
   - 한국: POSCO (철강), 삼성SDI (배터리 원가 경쟁력)
   → 지표: 영업이익률이 동업종 평균 대비 1.5배 이상

5. 규제/특허 (Regulatory Moat)
   - 글로벌: 제약회사 (FDA 승인), 방산 기업
   - 한국: 한국전력 (독점), 셀트리온 (바이오시밀러 특허)
   → 지표: 진입 장벽의 법적/기술적 높이
```

#### 전략 2: 집중 투자 + 장기 보유

```
버핏의 포트폴리오 구성 원칙:
  - 상위 5개 종목 = 전체 포트폴리오의 70% 이상
  - "20개의 펀치 카드" 원칙:
    평생 20번만 투자할 수 있다면, 매번 신중해진다

보유 기간:
  - 코카콜라: 1988년 매수 → 37년+ 보유 중 (배당 수익만으로 원금 회수)
  - American Express: 1964년 매수 → 62년+ 보유 중
  - 애플: 2016년 매수 → 버크셔 최대 보유 종목

한국 시장 적용:
  - 집중 투자 적용 시 최소 3종목, 최대 10종목
  - 한 종목 비중 상한: 30% (한국 시장 변동성 감안)
  - 코스피 대형우량주 중심으로 5년 이상 보유 목표

  예시 포트폴리오 (중급 투자자, 3,000만원):
  삼성전자  30% (900만원) - 반도체 해자
  네이버    20% (600만원) - 플랫폼 해자
  POSCO홀딩스 15% (450만원) - 비용 우위 해자
  현대차    15% (450만원) - 브랜드 + 전환비용
  현금      20% (600만원) - 추가 매수 대기
```

#### 전략 3: 내재가치 계산 (DCF 간이법)

```
정밀 DCF:
  내재가치 = SUM[ FCF(t) / (1 + r)^t ] + 잔존가치 / (1 + r)^n

  FCF = 영업현금흐름 - 자본적지출(CAPEX)
  r   = 할인율 (보통 8~12%)
  n   = 예측 기간 (보통 10년)

간이 버핏 공식 (초보자용):
  적정 PER = 이익 성장률 x 2

  예시:
  A기업 - 순이익 성장률 15%, 현재 PER 20
  적정 PER = 15 x 2 = 30
  현재 PER 20 < 적정 PER 30 → 저평가 (매수 고려)

  B기업 - 순이익 성장률 5%, 현재 PER 25
  적정 PER = 5 x 2 = 10
  현재 PER 25 > 적정 PER 10 → 고평가 (매수 보류)

Owner Earnings (주주이익):
  = 순이익 + 감가상각비 - 유지보수 CAPEX
  → 실질적으로 주주에게 귀속되는 현금
```

### 실행 체크리스트

**사업 품질**:
- [ ] 10년 후에도 살아남을 기업인가?
- [ ] 사업 모델을 초등학생에게 설명할 수 있는가?
- [ ] 일관된 운영 이력이 10년 이상인가?
- [ ] 경제적 해자가 1개 이상 존재하는가?

**경영진 품질**:
- [ ] 자본 배분 능력이 합리적인가? (ROE 추이로 확인)
- [ ] 주주에게 솔직하게 소통하는가? (IR 자료, 주주서한)
- [ ] 자사주 매입이나 배당으로 주주환원을 하는가?

**재무 지표**:
- [ ] ROE > 15% (최근 5년 평균)
- [ ] 부채비율 < 50% (무차입 경영 선호)
- [ ] 영업이익률 > 15% (가격 결정력 보유 의미)
- [ ] FCF(잉여현금흐름) 5년 연속 양수
- [ ] 매출 성장률 > 인플레이션율 (실질 성장)

### 초보자를 위한 시작 가이드

**Step 1**: 버크셔 해서웨이 주주서한 최근 5년치 읽기 (무료, berkshirehathaway.com)
**Step 2**: 본인이 10년 이상 사용할 제품/서비스를 만드는 기업 5개 선정
**Step 3**: 각 기업의 경제적 해자 유형을 파악하고 ROE, 영업이익률 확인
**Step 4**: 내재가치 간이 계산 후 현재 주가와 비교 (안전마진 20%+ 확보)
**Step 5**: 확신 있는 1~2개 종목에 집중 투자, 최소 3년 보유 계획 수립

### 대표 투자 성과

- **버크셔 해서웨이 (1965~2024)**: 연평균 복합수익률 19.8% (S&P 500: 10.2%)
- **60년 누적**: 1965년 $1,000 투자 시 → 약 $43,000,000 (4만 3천배)
- **코카콜라 투자**: 1988년 $1.3B 매수 → 현재 시가 $25B+ (배당 수익만 연 $700M+)
- **애플 투자**: 2016년부터 매수 → 2024년 기준 버크셔 전체 포트폴리오의 약 45%

## 3. 찰리 멍거 (Charlie Munger) - 다학제적 사고의 거장

**적합 프로필**:
| 항목 | 내용 |
|------|------|
| 레벨 | 중급~고급 |
| 리스크 성향 | 전 범위 (사고 방식 훈련이 핵심) |
| 최소 자본 | 자본 무관 (투자 철학/마인드셋 중심) |
| 추천 대상 | 투자 사고력을 체계적으로 훈련하고 싶은 분, 인지 편향을 극복하고 싶은 분, 다양한 분야 학습을 즐기는 분 |

**대표 저서**: <<불쌍한 찰리의 연감>> (Poor Charlie's Almanack), <<멍거의 투자 철학>> (Tao of Charlie Munger)

### 핵심 철학
> "평범한 기업을 놀라운 가격에 사는 것보다, 훌륭한 기업을 합리적인 가격에 사는 것이 낫다. 그리고 그것을 오래 보유하라."

### 투자 전략

#### 전략 1: 멘탈 모델 격자 (Latticework of Mental Models)

```
핵심 멘탈 모델 12가지 (투자 의사결정에 직접 활용):

1. 복리 효과 (수학)
   연 10% x 30년 = 17.4배 / 연 15% x 30년 = 66.2배
   → 수익률 5% 차이가 30년 뒤 4배 차이

2. 기회비용 (경제학)
   "이 주식을 사면 S&P 500 ETF보다 나은가?"
   → 모든 투자를 인덱스 대비로 비교

3. 인센티브 구조 (심리학)
   경영진 보상 체계가 주주 이익과 일치하는가?
   → 스톡옵션 과다 = 단기 주가 부양 유인

4. 확증 편향 (심리학)
   내가 좋아하는 정보만 찾고 있지 않은가?
   → 반드시 "이 투자가 실패하는 시나리오"를 먼저 생각

5. 생존자 편향 (통계학)
   성공 사례만 보고 판단하지 않는가?
   → 같은 전략으로 실패한 사례도 함께 분석

6. 임계 질량 (물리학)
   네트워크 효과가 임계점을 넘었는가?
   → 카카오톡, 쿠팡 등: 넘은 후 경쟁자 진입 극히 어려움

7. 평균 회귀 (통계학)
   지나치게 높은 성장률은 결국 평균으로 돌아온다
   → PER 50+ 성장주의 장기 수익률 주의

8. 역선택 (경제학)
   왜 이 주식이 이렇게 싼가? 내가 모르는 것이 있는가?
   → 저평가에는 반드시 이유가 존재

9. 규모의 경제 (경제학)
   생산량 증가 → 단위 비용 감소
   → 삼성전자 반도체, 현대차 전기차 등

10. 안전마진 (공학)
    다리를 지을 때 10톤 트럭이 다닌다면 30톤 기준으로 설계
    → 투자도 최악의 시나리오에서 버틸 수 있어야 함

11. 2차 효과 (시스템 사고)
    금리 인상 → 부동산 하락 → 건설사 실적 악화 → 은행 부실 대출 증가
    → 1차 효과만 보지 말고 연쇄 반응을 추적

12. 능력의 원 (자기인식)
    자신이 잘 아는 영역에서만 투자
    → "내가 모른다는 것을 아는 것"이 가장 중요
```

#### 전략 2: 역행 사고 (Inversion)

```
방법:
  "어떻게 하면 투자에서 망할까?"를 먼저 생각한다

투자 실패 원인 TOP 7 (역행 사고 체크리스트):
  1. 과도한 레버리지 (빚투)
     → 신용/미수 매수 절대 금지 (초보자)
  2. 모르는 것에 투자 (암호화폐, 파생상품 무지 상태 진입)
     → 설명할 수 없으면 투자하지 않는다
  3. 군중 심리 추종 (테마주, 급등주 추격 매수)
     → "모두가 사는 주식"은 이미 고평가
  4. 손실 회피 편향 (손절 못 함)
     → 매수 시점에 미리 손절 기준 설정
  5. 과잉 자신감 (단기 수익 후 베팅 규모 확대)
     → 연속 수익 = 실력이 아닐 수 있음
  6. 분산 부족 (한 종목에 올인)
     → 최소 3종목, 한 종목 비중 최대 30%
  7. 단기 매매 반복 (수수료 + 세금 누적)
     → 한국 주식 양도세(대주주), 미국 주식 22% 양도세 고려

한국 투자자 흔한 실패 패턴:
  - "이번엔 다르다" → 대부분 다르지 않음
  - "카페/유튜브 추천 종목" → 정보 비대칭의 피해자
  - "물타기" → 하락 종목에 추가 매수 = 손실 확대
  - "존버" → 근거 없는 장기 보유 ≠ 가치투자
```

#### 전략 3: 능력의 범위 (Circle of Competence)

```
실천 방법:

1단계 - 능력의 원 정의:
  내가 업무/취미/경험으로 잘 아는 산업은?

  예시:
  - IT 개발자 → 소프트웨어/SaaS 기업
  - 의료인 → 제약/바이오/의료기기
  - 유통업 종사자 → 리테일/물류
  - 자동차 마니아 → 완성차/부품업체

2단계 - 경계 인식:
  "나는 반도체 설계는 알지만, 장비/소재는 모른다"
  → 모르는 영역은 인덱스 ETF로 대체

3단계 - 점진적 확장:
  능력의 원을 넓히려면 최소 6개월 학습 필요
  - 해당 산업 보고서 월 2편 이상 읽기
  - IR 발표 자료 분기별 확인
  - 관련 전문가/종사자와 대화

한국 투자자 적용:
  잘 아는 분야           → 개별 종목 투자
  어느 정도 아는 분야    → 섹터 ETF 투자
  잘 모르는 분야         → 시장 전체 인덱스 ETF
```

### 실행 체크리스트

**사고 훈련**:
- [ ] 이 투자가 실패하는 시나리오 3가지를 적었는가? (역행 사고)
- [ ] 내 능력의 원 안에 있는 산업/기업인가?
- [ ] 확증 편향 없이 반대 의견도 찾아보았는가?
- [ ] 군중 심리에 의한 매수가 아닌가?

**의사결정 품질**:
- [ ] 최소 3개의 멘탈 모델을 적용하여 분석했는가?
- [ ] 기회비용을 고려했는가? (이 돈으로 인덱스를 사면 어떨까?)
- [ ] 2차, 3차 효과까지 생각했는가?
- [ ] 이 결정을 5년 뒤에도 자랑스러워할 수 있는가?

### 초보자를 위한 시작 가이드

**Step 1**: <<불쌍한 찰리의 연감>> 중 "인간 오판의 심리학" 강연 읽기
**Step 2**: 자신의 능력의 원(Circle of Competence) 지도를 그려본다 (직업, 취미, 경험 기반)
**Step 3**: 과거 투자 실패 경험을 역행 사고로 분석 (왜 실패했는가?)
**Step 4**: 매주 1가지 멘탈 모델을 학습하고 실제 투자 판단에 적용해 본다
**Step 5**: 투자 일지에 "근거"와 "반대 논거"를 반드시 함께 기록한다

### 대표 투자 성과

- **멍거 파트너십 (1962~1975)**: 연평균 24.3% (동기간 다우 지수 연 6.4%)
- **버크셔 공동 경영 (1978~2023)**: 버핏과 함께 연평균 19.8% 달성에 결정적 기여
- **코스트코 이사회**: 멍거의 추천으로 버크셔 투자, 코스트코 2000년대 이후 주가 15배+
- **BYD 투자 (2008)**: $230M 투자 → 2022년 피크 시가 $9B+ (약 39배)

## 4. 피터 린치 (Peter Lynch) - 생활 속 투자의 달인

**적합 프로필**:
| 항목 | 내용 |
|------|------|
| 레벨 | 입문~중급 |
| 리스크 성향 | 중위험 |
| 최소 자본 | 300만원 (분산 소액 투자 가능) |
| 추천 대상 | 개별 종목에 관심 있는 분, 일상 관찰력이 좋은 분, 성장주와 가치주를 균형 있게 접근하고 싶은 분 |

**대표 저서**: <<전설로 떠나는 월가의 영웅>> (One Up on Wall Street), <<피터 린치의 이기는 투자>> (Beating the Street)

### 핵심 철학
> "일상생활에서 훌륭한 투자 아이디어를 찾을 수 있다. 당신이 다니는 쇼핑몰, 직장, 식당에 10배 주식(Ten Bagger)이 숨어 있다."

### 투자 전략

#### 전략 1: PEG 비율 활용 (GARP 전략)

```
공식:
  PEG = PER / 연간 이익 성장률(%)

판단 기준:
  PEG < 0.5  → 매우 저평가 (강력 매수)
  PEG < 1.0  → 저평가 (매수)
  PEG 1.0~2.0 → 적정 가치
  PEG > 2.0  → 고평가 (매수 보류)

한국 시장 적용 예시:
  A전자 - PER 12, 이익 성장률 20%
  PEG = 12 / 20 = 0.6 → 저평가 (매수 고려)

  B바이오 - PER 80, 이익 성장률 30%
  PEG = 80 / 30 = 2.67 → 고평가 (매수 보류)

  C은행 - PER 5, 이익 성장률 8%
  PEG = 5 / 8 = 0.625 → 저평가 (매수 고려)

주의:
  - 이익 성장률은 최근 3~5년 평균 사용
  - 일시적 실적 급등/급락은 제외
  - 적자 기업은 PEG 적용 불가
```

#### 전략 2: 일상에서 투자 아이디어 발굴

```
린치의 10배 주식(Ten Bagger) 발굴법:

관찰 영역:
  1. 쇼핑: 자주 가는 매장, 인기 브랜드
  2. 직장: 업계 트렌드, 유망 협력사
  3. 취미: 새로 뜨는 제품/서비스
  4. 뉴스: 규제 변화, 기술 혁신

한국에서의 Ten Bagger 사례:
  - 셀트리온 (2012 → 2020): 바이오시밀러 성장, 약 15배
  - 카카오 (2014 → 2021): 모바일 플랫폼 확장, 약 12배
  - 에코프로비엠 (2020 → 2023): 2차전지 소재, 약 20배+
  - 삼성바이오로직스 (2016 → 2021): CDMO 성장, 약 8배

일상 관찰 → 투자로 전환하는 3단계:
  1단계 - 관찰: "요즘 이 서비스/제품이 대세다"
  2단계 - 확인: 실제 매출/이익이 성장하고 있는가?
  3단계 - 검증: PEG, 부채비율, 경쟁 환경 분석
```

#### 전략 3: 6가지 주식 분류와 전략

```
분류별 한국 시장 예시 및 전략:

1. 저성장주 (Slow Growers) - 성장률 2~4%
   한국: 한국전력, KT, SK텔레콤
   전략: 배당 수익 목적, PER 10 이하에서만 매수
   기대수익: 연 5~7% (배당 포함)

2. 우량주 (Stalwarts) - 성장률 10~12%
   한국: 삼성전자, LG생활건강, KT&G
   전략: 30~50% 수익 시 일부 매도, 하락 시 재매수
   기대수익: 연 10~15%

3. 고성장주 (Fast Growers) - 성장률 20%+
   한국: 2차전지 소재주, AI/반도체 장비주, 바이오
   전략: PEG < 1.5 확인 후 매수, 성장 둔화 시 매도
   기대수익: 연 20~30% (Ten Bagger 가능)
   주의: 성장률 25% 이상이 3년 넘게 지속되기 어려움

4. 경기순환주 (Cyclicals) - 경기에 연동
   한국: 현대차, 포스코, 대한항공, 호텔신라
   전략: 경기 저점(PER 높을 때가 오히려 매수 시점!)에 매수
   핵심: PER이 낮아 보일 때가 매도 시점 (이익 정점)

5. 회생주 (Turnarounds) - 위기에서 회복
   한국: 대우조선해양(한화오션), 쌍용차(KG모빌리티)
   전략: 부채 구조조정 완료 + 흑자 전환 확인 후 매수
   기대수익: 2~5배 (단, 고위험)

6. 자산주 (Asset Plays) - 숨은 자산 보유
   한국: 지주회사(지분가치 할인), 부동산 보유 기업
   전략: 순자산가치(NAV) 대비 50% 이상 할인 시 매수
   예시: 시가총액 1,000억 < 보유 부동산 + 지분 가치 2,000억
```

### 실행 체크리스트

**종목 발굴**:
- [ ] 일상에서 관찰한 아이디어인가? (남의 추천이 아니라 나의 관찰)
- [ ] 이 기업의 사업을 2분 안에 설명할 수 있는가?
- [ ] 6가지 분류 중 어디에 해당하는가?

**재무 검증**:
- [ ] PEG < 1.5 (성장주의 경우)
- [ ] 부채비율 < 80%
- [ ] 최근 3년 매출 성장률 양수
- [ ] 내부자(경영진/임원) 매수 이력이 있는가?

**매도 기준**:
- [ ] 성장 스토리가 변했는가? (원래 매수 이유가 사라졌는가?)
- [ ] PEG > 2.0으로 상승했는가?
- [ ] 더 좋은 PEG 종목을 발견했는가?

### 초보자를 위한 시작 가이드

**Step 1**: <<전설로 떠나는 월가의 영웅>> 읽기 (특히 6가지 분류법 챕터)
**Step 2**: 일상에서 좋아하는 제품/서비스 10개를 적고, 해당 기업을 조사
**Step 3**: 각 기업의 PEG 비율을 계산하여 1.0 미만인 종목 필터링
**Step 4**: 6가지 분류법으로 각 종목을 분류하고 해당 전략 적용
**Step 5**: 포트폴리오를 3~4개 분류에 걸쳐 분산 (고성장주 + 우량주 + 자산주 등)

### 대표 투자 성과

- **마젤란 펀드 (1977~1990)**: 연평균 29.2%, 13년간 운용
- **$18M → $14B**: 펀드 규모 약 777배 성장
- **S&P 500 대비**: 동기간 S&P 500 연평균 약 15% → 린치 2배 초과수익
- **Ten Bagger 다수 발굴**: Dunkin' Donuts, Taco Bell, The Limited, Fannie Mae 등

## 5. 필립 피셔 (Philip Fisher) - 성장주 투자의 선구자

**적합 프로필**:
| 항목 | 내용 |
|------|------|
| 레벨 | 중급~고급 |
| 리스크 성향 | 중~고위험 |
| 최소 자본 | 1,000만원 (집중 투자 특성상) |
| 추천 대상 | 기업 분석에 깊은 관심이 있는 분, 리서치에 시간을 투자할 수 있는 분, 성장주 장기 투자를 원하는 분 |

**대표 저서**: <<위대한 기업에 투자하라>> (Common Stocks and Uncommon Profits, 1958)

### 핵심 철학
> "평범한 회사의 주식을 싼 가격에 사는 것보다, 뛰어난 회사의 주식을 적정 가격에 사는 것이 훨씬 낫다. 뛰어난 기업의 주식을 사서 영원히 보유하라."

### 투자 전략

#### 전략 1: 15가지 질문법 (기업 분석 프레임워크)

```
성장 잠재력 (질문 1~3):
  Q1. 향후 수년간 매출을 크게 늘릴 제품/서비스가 있는가?
      한국 적용: 삼성전자의 HBM/AI 반도체, 셀트리온의 신규 바이오시밀러

  Q2. 현재 성장동력이 소진되면 새 제품을 개발할 역량이 있는가?
      확인법: R&D 투자 비중(매출 대비 5% 이상), 특허 출원 수

  Q3. R&D 노력이 기업 규모 대비 효과적인가?
      지표: R&D 투자 대비 신제품 매출 비중

운영 효율성 (질문 4~10):
  Q4.  평균 이상의 영업 조직을 갖추고 있는가?
  Q5.  영업이익률이 높은가? (동업종 상위 25% 이내)
  Q6.  이익률을 유지/개선하기 위한 구체적 노력이 있는가?
  Q7.  우수한 노사 관계를 유지하고 있는가? (이직률, 파업 이력)
  Q8.  경영진 간 관계가 원활한가?
  Q9.  경영 깊이(Depth)가 있는가? (CEO 외 2인자, 3인자 역량)
  Q10. 원가 분석과 회계 관리가 정밀한가?

경영진/경쟁 우위 (질문 11~15):
  Q11. 산업 내 독보적 경쟁 우위가 있는가?
  Q12. 장기적 관점에서 이익을 추구하는가? (단기 실적 부풀리기 아닌가)
  Q13. 성장 자금 조달 시 주식 희석이 과도하지 않은가?
       확인법: 최근 5년간 유상증자/전환사채 발행 빈도
  Q14. 나쁜 소식도 솔직히 공유하는가? (IR 투명성)
  Q15. 경영진의 도덕성에 의심이 없는가?

판정:
  15개 중 12개 이상 Yes → 강력 매수 후보
  15개 중 9~11개 Yes → 추가 조사 필요
  15개 중 8개 이하 Yes → 투자 부적합
```

#### 전략 2: Scuttlebutt 리서치 (탐문 조사법)

```
정보원 5가지와 활용법:

1. 고객 (해당 기업의 제품/서비스 이용자)
   질문: "이 제품의 장단점은? 재구매 의향은?"
   한국 적용: 네이버 카페, 블로그 후기, 앱 리뷰 분석

2. 경쟁사 직원/관계자
   질문: "이 회사를 경쟁자로서 어떻게 평가하는가?"
   한국 적용: 잡플래닛/블라인드 경쟁사 평가, 업계 세미나

3. 공급업체
   질문: "이 회사와의 거래는 안정적인가? 물량 추세는?"
   한국 적용: 공급망 분석 (ex: 삼성전자 → 부품사 동향)

4. 전/현직 직원
   질문: "회사 문화는? 경영진 역량은? 성장 가능성은?"
   한국 적용: 블라인드, 잡플래닛, 크레딧잡 리뷰

5. 산업 전문가/연구자
   질문: "이 산업의 향후 5~10년 전망은?"
   한국 적용: 증권사 애널리스트 리포트 (네이버 증권 리서치)

개인 투자자가 Scuttlebutt를 실천하는 현실적 방법:
  - 네이버/다음 카페에서 제품 사용 후기 100개+ 분석
  - 블라인드/잡플래닛에서 해당 기업 직원 평가 확인
  - 유튜브에서 전문가/애널리스트 분석 영상 3개+ 시청
  - 분기별 실적발표 컨퍼런스콜 직접 청취 (DART 공시)
  - 주주총회 참석 (개인주주도 가능)
```

#### 전략 3: 집중 투자 + 초장기 보유

```
피셔의 포트폴리오 원칙:
  - 최대 5~10개 종목만 보유
  - "10배 오를 주식 1개가 -50% 종목 5개를 상쇄"
  - 매도 조건은 극히 제한적

피셔의 3가지 매도 조건 (이것만 아니면 보유):
  1. 최초 분석이 틀렸음을 인정할 때
  2. 기업이 더 이상 성장 기준을 충족하지 못할 때
  3. 훨씬 더 뛰어난 투자 기회를 발견했을 때

매도하지 말아야 할 경우:
  - 단기 실적 부진 (성장 스토리가 유효하다면)
  - 시장 전체 하락 (개별 기업 문제가 아니라면)
  - 주가가 "너무 올랐다"는 느낌 (감으로 매도 금지)

한국 시장 적용:
  매수 후 최소 보유 기간: 3년 (성장 확인)
  이상적 보유 기간: 5~10년+
  리밸런싱: 연 1회 (15가지 질문 재점검)
```

### 실행 체크리스트

**Scuttlebutt 리서치**:
- [ ] 해당 기업 제품/서비스를 직접 사용해 보았는가?
- [ ] 고객 리뷰 최소 50개 이상 분석했는가?
- [ ] 경쟁사 대비 강점/약점을 파악했는가?
- [ ] 블라인드/잡플래닛에서 직원 평가를 확인했는가?
- [ ] 최근 2~3년 실적발표 컨퍼런스콜을 들어보았는가?

**15가지 질문**:
- [ ] 15개 질문 중 12개 이상 "Yes"인가?
- [ ] 성장 잠재력(Q1~Q3) 전부 "Yes"인가?
- [ ] 경영진 정직성(Q14~Q15)에 의심이 없는가?

### 초보자를 위한 시작 가이드

**Step 1**: <<위대한 기업에 투자하라>> 읽기 (특히 15가지 질문 챕터)
**Step 2**: 관심 종목 1개를 선정하고 15가지 질문 체크리스트를 작성
**Step 3**: Scuttlebutt 리서치 실습 - 해당 기업 제품 사용 + 리뷰 분석 + 직원 평가 확인
**Step 4**: DART에서 최근 사업보고서, IR 자료를 읽고 성장성 판단
**Step 5**: 확신이 생기면 소액으로 매수, 분기별로 15가지 질문 재점검

- **피셔의 원칙 실현**: 소수 종목 집중 투자로 시장 대비 초과수익 수십 년간 달성
- **버핏에 대한 영향**: "나의 투자 철학은 85% 그레이엄 + 15% 피셔" (버핏 본인 언급)

## 6. 레이 달리오 (Ray Dalio) - 올웨더의 설계자

**적합 프로필**:
| 항목 | 내용 |
|------|------|
| 레벨 | 중급 |
| 리스크 성향 | 저~중위험 |
| 최소 자본 | 500만원 (다수 ETF 분산 투자) |
| 추천 대상 | 분산 투자를 선호하는 분, 변동성을 최소화하고 싶은 분, 거시경제에 관심 있는 분, 잠 편하게 자고 싶은 분 |

**대표 저서**: <<원칙>> (Principles, 2017), <<변화하는 세계 질서>> (Principles for Dealing with the Changing World Order, 2021)

### 핵심 철학
> "경제는 기계처럼 작동한다. 그 작동 원리를 이해하면 모든 경제 환경에서 수익을 낼 수 있다."

### 투자 전략

#### 전략 1: 올웨더 포트폴리오 (All Weather Portfolio)

```
경제 4가지 환경과 유리한 자산:

            인플레 상승          인플레 하락
          ┌─────────────────┬─────────────────┐
 경제     │ 원자재, 금,      │ 주식,           │
 성장     │ 물가연동채권,    │ 회사채,          │
          │ 신흥국 주식      │ 성장주           │
          ├─────────────────┼─────────────────┤
 경제     │ 금,             │ 장기 국채,       │
 침체     │ 물가연동채권    │ 현금             │
          └─────────────────┴─────────────────┘

올웨더 배분 비율:
  주식         30%
  장기 국채    40%
  중기 국채    15%
  금           7.5%
  원자재       7.5%

한국 투자자용 ETF 포트폴리오 (500만원 기준):
  TIGER 미국S&P500     150만원 (30%)  → 주식
  KODEX 미국채울트라30년선물  200만원 (40%)  → 장기채
  TIGER 미국채10년선물   75만원 (15%)  → 중기채
  KODEX 골드선물(H)      37.5만원 (7.5%)  → 금
  KODEX WTI원유선물(H)   37.5만원 (7.5%)  → 원자재

미국 ETF 직접 투자 버전:
  VTI (미국 전체 주식)     30%
  TLT (미국 장기 국채)     40%
  IEF (미국 중기 국채)     15%
  GLD (금)                 7.5%
  DJP (원자재)             7.5%

리밸런싱: 반기 1회 또는 비중 5% 이상 이탈 시
```

#### 전략 2: 리스크 패리티 (Risk Parity)

```
전통적 60/40 포트폴리오의 문제:
  주식 60% + 채권 40%
  → 리스크의 약 90%가 주식에서 발생
  → "분산 투자"라 하지만 실질적으로 주식에 의존

리스크 패리티 개념:
  각 자산군의 "리스크 기여도"를 동일하게 맞춤

  리스크 기여도 = 자산 비중 x 자산 변동성 x 상관계수

  채권은 변동성이 낮으므로 → 비중을 높임 (레버리지 사용)
  주식은 변동성이 높으므로 → 비중을 낮춤

간이 리스크 패리티 (레버리지 없이):
  연간 변동성 기준:
  - 주식: 약 15~20%
  - 채권: 약 5~8%
  - 금: 약 12~15%
  - 원자재: 약 18~22%

  리스크 균등화 비중 (대략):
  - 주식: 20%
  - 채권: 50%
  - 금: 15%
  - 원자재: 15%

한국 투자자 간이 적용법:
  코스피 ETF         20%
  미국채 ETF         30%
  한국채 ETF         20%
  금 ETF             15%
  달러 자산(예금)    15%

  → 단순하지만 리스크 분산 효과 확보
```

#### 전략 3: 경제 기계 이론 (How The Economic Machine Works)

```
달리오의 경제 3대 동력:

1. 생산성 성장 (장기 추세선, 연 2~3%)
   - 기술 발전, 교육, 인프라
   - 50~100년 단위의 완만한 상승

2. 단기 부채 사이클 (5~8년)
   - 경기 순환의 핵심
   - 금리 인하 → 대출 증가 → 경기 과열 → 금리 인상 → 경기 둔화

   한국 사례:
   2020: 코로나 → 금리 0.5% → 유동성 폭발
   2022: 인플레 → 금리 3.5% → 부동산/주식 조정
   2024~25: 금리 인하 사이클 진입 → 자산 가격 반등

3. 장기 부채 사이클 (50~75년)
   - 부채가 소득 대비 감당할 수 없는 수준에 도달
   - 디레버리징(부채 축소) 발생

   역사적 사례:
   1930년대 대공황
   2008년 금융위기
   일본 잃어버린 30년 (1990~)

현재 위치 파악 (투자 타이밍):
  단기 사이클 초기 (금리 인하기): 주식, 부동산 비중 확대
  단기 사이클 후기 (금리 인상기): 현금, 단기채 비중 확대
  장기 사이클 말기 (과잉 부채): 금, 물가연동채 비중 확대
```

### 실행 체크리스트

**포트폴리오 구성**:
- [ ] 4가지 경제 환경 각각에 대응하는 자산이 포함되어 있는가?
- [ ] 어떤 단일 자산의 비중도 40%를 넘지 않는가?
- [ ] 자산 간 상관관계가 낮은 조합인가?
- [ ] 리밸런싱 주기를 정했는가? (반기 1회 권장)

**거시경제 모니터링**:
- [ ] 현재 단기 부채 사이클의 위치를 파악했는가?
- [ ] 중앙은행(한국은행/Fed) 금리 방향을 확인했는가?
- [ ] 인플레이션율 추이를 확인했는가?
- [ ] VIX(공포 지수)가 30 이상인지 확인했는가?

### 초보자를 위한 시작 가이드

**Step 1**: 유튜브 "How The Economic Machine Works by Ray Dalio" 시청 (30분, 한글 자막)
**Step 2**: 올웨더 포트폴리오의 한국 ETF 버전으로 포트폴리오 구성

**Step 4**: 6개월마다 리밸런싱 실행 (비중 조정)

### 대표 투자 성과

## 7. 조지 소로스 (George Soros) - 매크로 트레이딩의 제왕

**적합 프로필**:
| 항목 | 내용 |
|------|------|
| 레벨 | 고급 |
| 리스크 성향 | 고위험 |
| 최소 자본 | 5,000만원+ (매크로 베팅 특성상 충분한 자본 필요) |
| 추천 대상 | 거시경제 분석 능력이 있는 분, 높은 변동성을 감내할 수 있는 분, 금융시장 전반(환율/금리/원자재)에 대한 이해가 있는 분 |

**대표 저서**: <<금융의 연금술>> (The Alchemy of Finance, 1987), <<소로스의 투자 특강>> (Soros on Soros)

### 핵심 철학
> "시장은 항상 편향되어 있다. 우리의 세계 인식은 항상 불완전하고 왜곡되어 있으며, 이 왜곡된 인식이 시장 가격에 반영되고, 그 가격이 다시 현실을 변화시킨다."

### 투자 전략

#### 전략 1: 재귀성 이론 (Reflexivity)

```
전통 경제학:
  펀더멘털 → 가격 (일방향)
  "시장은 효율적이다"

소로스의 재귀성:
  인식 ⇄ 현실 (양방향 피드백 루프)
  "시장은 효율적이지 않다. 시장 참여자의 인식이 현실을 변화시킨다"

재귀성의 작동 메커니즘:

  상승 재귀 (버블 형성):
  주가 상승 → 기업 담보가치 증가 → 대출 용이 → 투자 확대
  → 실적 개선 → 주가 추가 상승 → ... (자기강화 루프)

  하락 재귀 (버블 붕괴):
  주가 하락 → 담보가치 감소 → 대출 회수 → 투자 축소
  → 실적 악화 → 주가 추가 하락 → ... (악순환)

한국 시장 재귀성 사례:

  2020~2021 "동학개미" 현상:
  코로나 폭락 → 개인 매수 → 주가 반등 → 성공 경험 확산
  → 더 많은 신규 투자자 유입 → 주가 추가 상승 → 코스피 3300
  → 과열 → 2022년 조정

  부동산 재귀성:
  아파트 가격 상승 → "영끌" 심리 → 수요 증가 → 가격 추가 상승
  → 정부 규제 → 수요 위축 → 가격 하락 → 패닉셀 → 추가 하락

활용법:
  1. 현재 어떤 재귀 루프가 작동 중인지 파악
  2. 자기강화 루프의 한계점(임계점) 예측
  3. 임계점 근처에서 반대 포지션 진입
```

#### 전략 2: 매크로 트레이딩 (Global Macro)

```
소로스의 매크로 분석 프레임워크:

분석 대상:
  1. 통화 (Currency)

```

```
  ┌─────────┬──────────┬──────────┬──────────┐

한국 적립식 투자 실행 방법:
  1. 증권사 앱에서 "자동 매수" 설정
```

## 투자 대가들의 공통 원칙

### 원칙 1: 원금 보전이 최우선이다

```
그레이엄: "안전마진 30% 이상 확보"
버핏:    "Rule No.1: 절대 돈을 잃지 마라"
달리오:  "리스크 패리티로 최대 낙폭 제한"
존 리:   "분산 투자 + 적립식으로 리스크 분산"

→ 수익보다 손실 방지가 먼저다
→ 원금이 50% 줄면 원금 회복에 100% 수익이 필요하다
   (100만원 → 50만원 → 100만원 회복에 +100% 필요)
```

### 원칙 2: 장기적 관점을 유지한다

```
버핏:  "최고의 보유 기간은 영원이다"
피셔:  "훌륭한 기업은 10년 이상 보유"
린치:  "3~5년 보유로 10배 수익 달성"
존 리: "20~30년 적립식 투자"

→ 단기 매매는 투기, 장기 보유가 투자
→ 복리는 시간이 길수록 폭발적으로 작용
→ 한국 투자자의 평균 주식 보유 기간 약 2개월 → 이것이 손실의 원인
```

### 원칙 3: 이해하는 것에만 투자한다

```
버핏:  "내가 이해하지 못하면 투자하지 않는다"
멍거:  "능력의 원 밖으로 나가지 마라"
린치:  "아는 것에 투자하라"
그레이엄: "투기와 투자를 구분하라"

→ "이 회사가 뭘 하는지 설명할 수 없으면 사지 마라"
```

```
멍거:  "인지 편향을 인식하고 극복하라"
소로스: "틀렸으면 즉시 인정하고 손절하라"
```

```
버핏:  "하루 5~6시간 독서" (80대에도)
멍거:  "평생 학습 기계가 되라"
달리오: "진실을 추구하라"
린치:  "개인 투자자의 최대 무기는 학습"

→ 투자는 평생 학습의 여정
→ 시장은 계속 변하지만 원칙은 변하지 않는다
→ 1년에 최소 투자 서적 3권, 연간 보고서 10개 이상 읽기
→ 실수에서 가장 많이 배운다 (투자 일지 필수)
```

## 레벨별 추천 학습 순서

### 입문 (투자 경험 0~2년)

```
학습 경로:
  그레이엄(방어적 투자자) → 존 리 → 피터 린치

1단계 - 그레이엄 방어적 투자자 (1~3개월):
  핵심: 안전마진 개념 이해, PER/PBR 기초
  실행: 인덱스 ETF 매수 시작 (KODEX 200, TIGER S&P500)
  목표: "투자 ≠ 투기" 마인드 확립

2단계 - 존 리 적립식 투자 (3~6개월):
  핵심: 복리의 마법, DCA(적립식) 습관화
  실행: 매월 자동 매수 시스템 구축 (ISA/연금저축 활용)
  목표: 6개월간 한 번도 빠지지 않고 적립 완수

3단계 - 피터 린치 개별 종목 입문 (6~12개월):
  핵심: 일상 관찰 → 투자 아이디어 발굴, PEG 활용
  실행: 관심 종목 3~5개 선정 → 소액 매수 → 6가지 분류 적용
  목표: 개별 종목 분석의 기초 역량 확보
```

### 중급 (투자 경험 2~5년)

```
학습 경로:
  버핏 → 달리오 → 피셔

1단계 - 버핏 기업 분석 심화 (1~6개월):
  핵심: 경제적 해자, DCF, 집중 투자
  실행: 보유 종목의 해자 분석, 내재가치 계산 연습
  목표: "왜 이 기업인가"를 논리적으로 설명할 수 있는 수준

2단계 - 달리오 자산배분 (6~12개월):
  핵심: 올웨더, 리스크 패리티, 거시경제 사이클
  실행: 포트폴리오에 채권/금/원자재 추가 → 분산 투자 완성
  목표: 어떤 경제 환경에서도 흔들리지 않는 포트폴리오 구축

3단계 - 피셔 성장주 분석 (12~18개월):
  핵심: 15가지 질문법, Scuttlebutt 리서치
  실행: 관심 기업에 15가지 질문 적용, 심층 리서치 실습
  목표: "탁월한 기업"을 스스로 발굴할 수 있는 역량
```

### 고급 (투자 경험 5년+)

```
학습 경로:
  멍거 → 소로스 → 자신만의 투자 시스템 구축

1단계 - 멍거 사고력 훈련 (상시):
  핵심: 멘탈 모델 격자, 역행 사고, 인지 편향 극복
  실행: 투자 의사결정에 최소 3가지 멘탈 모델 적용
  목표: 다학제적 사고로 투자 판단의 품질 향상

2단계 - 소로스 매크로 관점 (필요시):
  핵심: 재귀성 이론, 글로벌 매크로 분석, 포지션 사이징
  실행: 거시경제 시나리오 작성, 환율/금리 변동 분석
  목표: 시장의 "큰 그림"을 읽는 능력 확보
  주의: 레버리지 매매는 전문가가 아닌 한 비추천

3단계 - 자신만의 투자 시스템 구축 (지속):
  핵심: 8인의 대가 철학 중 자신에게 맞는 요소 조합
  실행: 투자 원칙 문서 작성, 체크리스트 정립, 투자 일지 운영
  목표: 어떤 시장 환경에서도 일관된 성과를 내는 시스템

  시스템 구축 예시:
  "나의 투자 시스템"
  1. 종목 발굴: 린치식 일상 관찰 + 피셔식 15가지 질문
  2. 가치 평가: 그레이엄식 안전마진 + 버핏식 DCF
  3. 포트폴리오: 달리오식 자산배분 + 버핏식 집중 투자
  4. 사고 검증: 멍거식 역행 사고 + 멘탈 모델
  5. 리스크 관리: 소로스식 손절 원칙 + 포지션 사이징
  6. 실행 루틴: 존 리식 적립식 매수 + 분기별 리밸런싱
```

## 대가별 빠른 매칭 가이드

```
사용자 질문 → 매칭 대가:

"예금밖에 안 해봤는데 주식 시작하고 싶어요"
  → 존 리 (적립식 ETF) + 그레이엄 (방어적 투자자)

"개별 종목에 투자하고 싶은데 어떤 기업을 사야 할지 모르겠어요"
  → 린치 (일상 관찰, PEG) + 그레이엄 (안전마진)

"좋은 기업을 찾아서 오래 보유하고 싶어요"
  → 버핏 (해자, 장기보유) + 피셔 (15가지 질문)

"포트폴리오를 어떻게 구성해야 할지 모르겠어요"
  → 달리오 (올웨더, 자산배분)

"투자 판단력을 키우고 싶어요"
  → 멍거 (멘탈 모델, 역행 사고)

"환율이나 금리 같은 매크로 요인으로 투자하고 싶어요"
  → 소로스 (재귀성, 매크로) - 단, 고급 투자자만

"자녀에게 경제 교육을 해주고 싶어요"
  → 존 리 (자녀 증여, 금융교육)

"리스크를 최소화하면서 안정적으로 수익을 내고 싶어요"
  → 그레이엄 (안전마진) + 달리오 (올웨더)
```
