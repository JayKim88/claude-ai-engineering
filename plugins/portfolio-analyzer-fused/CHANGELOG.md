## Development Notes

### Architecture Philosophy
This plugin follows the **Competitive-Agents Fusion** approach:
- **From Alpha (Pragmatist)**: 9 working Python scripts, practical functionality
- **From Beta (Architect)**: 6-agent multi-tier design, cost optimization, comprehensive SKILL.md

### Model Usage
- **Opus 4.6**: 2 agents (strategic-advisor, portfolio-advisor) for high-quality reasoning
- **Sonnet 4.5**: 4 agents (market-analyst, opportunity-scanner, risk-assessor, data-manager) for cost efficiency

### Key Design Decisions
1. **Agent-Script Separation**: AI agents orchestrate, Python scripts compute
2. **Multi-Source Fallback**: Reliability through MCP ‚Üí yfinance ‚Üí pykrx chain
3. **Weighted Scoring**: 40% financial, 35% valuation, 25% momentum
4. **Cost Optimization**: Strategic Opus placement (2 of 6 agents)
