## 📈 가치투자 (Value Investing)

### 1. 벤저민 그레이엄 (Benjamin Graham) - 가치투자의 아버지

**대표 저서**: 《현명한 투자자》, 《증권 분석》

#### 핵심 철학
> "주식은 기업의 일부이다. 시장의 변동성을 활용하되, 지배당하지 마라."

#### 투자 전략

##### 1.1 안전마진 (Margin of Safety) ⭐⭐⭐⭐⭐
```
내재가치 - 시장가격 = 안전마진

원칙: 안전마진이 최소 30% 이상일 때만 매수
예시: 내재가치 $100인 기업을 $70 이하에 매수
```

**Market-Pulse 활용**:
- UsStockInfo MCP로 PER, PBR 확인
- 동종 업계 대비 저평가 종목 스크리닝

##### 1.2 Mr. Market 개념
```
시장 = 감정적인 사람 (Mr. Market)
- 때로는 과도하게 낙관적 (고평가)
- 때로는 과도하게 비관적 (저평가)

전략: Mr. Market의 비합리성을 이용
```

##### 1.3 그레이엄의 체크리스트

**정량적 기준**:
- [ ] PER < 15
- [ ] PBR < 1.5 (장부가 대비)
- [ ] 부채비율 < 100%
- [ ] 유동비율 > 2 (단기 채무 상환 능력)
- [ ] 이익 성장: 최근 10년 중 7년 이상 흑자

**정성적 기준**:
- [ ] 사업 모델 이해 가능
- [ ] 장기 존속 가능성
- [ ] 경영진 품질

#### 대표 투자 성과
- 1936-1956: 연평균 20% 수익률 (S&P 500 대비 2배)

### 2. 워렌 버핏 (Warren Buffett) - 오마하의 현인

**회사**: Berkshire Hathaway
**투자 기간**: 1965년 ~ 현재 (60년+)

#### 핵심 철학
> "Rule No. 1: Never lose money. Rule No. 2: Never forget Rule No. 1."

#### 투자 전략

##### 2.1 경제적 해자 (Economic Moat) ⭐⭐⭐⭐⭐
```
해자 = 경쟁 우위를 보호하는 장벽

해자 유형:
1. 브랜드 파워 (코카콜라, 애플)
2. 네트워크 효과 (Visa, Mastercard)
3. 비용 우위 (Costco, Walmart)
4. 전환 비용 (Microsoft, SAP)
5. 규제/특허 (제약회사, 철도)
```

**Market-Pulse 활용**:
- 섹터별 ROE 비교 (해자 있는 기업 = 높은 ROE)
- 10년+ 안정적 배당 기록 확인

##### 2.2 "가격은 당신이 지불하는 것, 가치는 당신이 얻는 것"

**버핏의 가치 계산법**:
```
기업 가치 = 미래 현금흐름의 현재가치

DCF (Discounted Cash Flow):
내재가치 = Σ(미래 FCF / (1+할인율)^n)

간단 버전:
PER < 15 + 이익 성장률 > 10% → 저평가
```

##### 2.3 집중 투자
```
버핏의 포트폴리오:
- 상위 5개 종목이 전체의 70%+ 차지
- "다각화는 무지를 보호하는 전략"

단, 초보자는 분산 권장!
```

##### 2.4 장기 보유
```
"최고의 보유 기간은 영원이다"

보유 기간:
- 코카콜라: 1988년 매수 → 36년+ 보유 중
- American Express: 1964년 매수 → 62년+ 보유 중
```

#### 버핏의 체크리스트

**사업 품질**:
- [ ] 10년 후에도 살아남을 기업?
- [ ] 단순하고 이해하기 쉬운 비즈니스?
- [ ] 일관된 운영 이력 (10년+)?
- [ ] 장기적으로 밝은 전망?

**경영진 품질**:
- [ ] 합리적인가? (자본 배분 능력)
- [ ] 주주에게 솔직한가?
- [ ] 군중 심리에 저항하는가?

**재무 지표**:
- [ ] ROE > 15% (10년 평균)
- [ ] 적은 부채 (부채비율 < 50%)
- [ ] 높은 이익률 (영업이익률 > 15%)
- [ ] 잉여 현금 흐름이 풍부한가?

#### 대표 투자 성과
- 1965-2023: 연평균 19.8% (S&P 500: 9.9%)
- $1,000 투자 → $4,384,000 (4,384배!)

### 3. 찰리 멍거 (Charlie Munger) - 버핏의 파트너

#### 핵심 철학
> "평범한 기업을 놀라운 가격에 사는 것보다, 훌륭한 기업을 합리적인 가격에 사는 것이 낫다."

#### 투자 전략

##### 3.1 다학제적 사고 (Mental Models)
```
투자 결정 = 여러 학문의 렌즈로 분석

Mental Models:
- 경제학: 공급/수요, 기회비용
- 심리학: 인지 편향, 군중 심리
- 수학: 확률, 복리
- 생물학: 진화, 생태계
- 물리학: 임계 질량, 관성
```

##### 3.2 역행 사고 (Inversion)
```
"내가 어떻게 하면 성공할까?" 대신
"내가 어떻게 하면 실패할까?"

실패 원인:
- 고레버리지 (과도한 차입)
- 사기 (부정 회계)
- 기술 변화 (디지털 카메라 vs 필름)
- 규제 리스크

→ 이를 피하면 성공!
```

##### 3.3 능력의 범위 (Circle of Competence)
```
이해하는 분야에만 투자

예시:
- 버핏: 보험, 소비재, 은행
- 멍거: 중국 전자상거래 (알리바바)

→ 모르는 분야는 피하라
```

#### 대표 투자 성과
- 1962-1975: 연평균 24.3% (자신의 펀드)

### 4. 피터 린치 (Peter Lynch) - GARP의 대가

**직함**: Fidelity Magellan Fund 매니저 (1977-1990)

#### 핵심 철학
> "일상 생활에서 훌륭한 투자 아이디어를 찾을 수 있다."

#### 투자 전략

##### 4.1 GARP (Growth At Reasonable Price) ⭐⭐⭐⭐⭐
```
성장주를 합리적인 가격에 매수

핵심 지표: PEG 비율
PEG = PER / 이익 성장률

판단:
- PEG < 1: 저평가 (매수)
- PEG 1-2: 적정
- PEG > 2: 고평가 (매도)

예시:
PER 20, 성장률 25% → PEG 0.8 → 매수!
```

**Market-Pulse 활용**:
- UsStockInfo MCP로 EPS 성장률 확인
- PEG < 1인 성장주 스크리닝

##### 4.2 "아는 것에 투자하라" (Invest What You Know)
```
린치의 10배 주식(Ten Baggers) 발굴법:

1. 일상에서 관찰
   - 던킨도너츠: 커피가 맛있다
   - Home Depot: DIY 열풍
   - Taco Bell: 빠른 서비스

2. 기본 조사
   - 사업 모델 이해
   - 재무제표 확인
   - 경쟁사 비교

3. 매수
   - PEG < 1.5
   - 부채비율 낮음
   - 이익 성장 지속
```

##### 4.3 린치의 6가지 주식 분류

**1. Slow Growers (저성장주)**
- 성장률: 2-4%
- 예: 전력 회사, 통신사
- 전략: 배당 수익 목적

**2. Stalwarts (우량주)**
- 성장률: 10-12%
- 예: 코카콜라, P&G
- 전략: 30-50% 수익 후 매도

**3. Fast Growers (고성장주)** ⭐
- 성장률: 20-25%+
- 예: 스타벅스, 아마존 초기
- 전략: 10배 수익 가능 (Ten Bagger)

**4. Cyclicals (경기순환주)**
- 예: 자동차, 항공
- 전략: 경기 저점 매수, 고점 매도

**5. Turnarounds (회생주)**
- 예: 크라이슬러 (1980년대)
- 전략: 고위험 고수익

**6. Asset Plays (자산주)**
- 예: 부동산, 자원 보유 기업
- 전략: 장부가 대비 저평가 매수

##### 4.4 린치의 13가지 체크리스트

**긍정 신호** (✅):
1. 회사명이 지루하거나 우스꽝스러움 (관심 덜 받음)
2. 지루한 사업 (남들이 무시)
3. 불쾌한 사업 (장례식장, 쓰레기 처리)
4. 스핀오프 (분사된 자회사)
5. 기관 투자자가 보유하지 않음
6. 소문이 안 좋음 (오해로 저평가)
7. 사람들이 싫어하는 산업
8. 무성장 산업
9. 니치 마켓 (틈새시장)
10. 사람들이 계속 사는 제품 (소모품)
11. 기술 활용 기업
12. 내부자 매수 (경영진이 자사주 매수)
13. 자사주 매입 프로그램

**부정 신호** (❌):
1. "차세대 ○○○" 소리
2. 다각화 (여러 사업)
3. 속삭임 주식 (루머)
4. 공급 과잉 산업
5. 대형 고객 의존도 높음
6. 멋진 이름

#### 대표 투자 성과
- 1977-1990: 연평균 29.2%
- 마젤란 펀드: $18M → $14B (13년간 777배!)

## 🚀 성장주 투자 (Growth Investing)

### 5. 필립 피셔 (Philip Fisher)

**대표 저서**: 《Common Stocks and Uncommon Profits》

#### 핵심 철학
> "평범한 회사의 주식을 싼 가격에 사는 것보다, 뛰어난 회사의 주식을 적정 가격에 사는 것이 낫다."

#### 투자 전략

##### 5.1 Scuttlebutt (소문 수집) 리서치 ⭐⭐⭐⭐
```
기업 조사 방법:

1. 고객 인터뷰
   - "이 회사 제품 어때요?"

2. 경쟁사 인터뷰
   - "경쟁사로서 이 회사를 어떻게 보나요?"

3. 공급업체 인터뷰
   - "이 회사와 거래하기 어때요?"

4. 전직 직원 인터뷰
   - "경영진 품질은?"

5. 연구기관 인터뷰
   - "이 산업의 미래는?"
```

##### 5.2 피셔의 15가지 질문

**성장 잠재력**:
1. 향후 수년간 매출을 크게 늘릴 제품/서비스가 있는가?
2. 현재 제품 라인이 소진되면 신제품 개발 능력이 있는가?
3. 연구개발 노력이 효과적인가?

**운영 효율성**:
4. 평균 이상의 판매 조직을 가지고 있는가?
5. 이익률이 높은가?
6. 이익률을 유지/개선하기 위해 노력하는가?
7. 탁월한 노사 관계를 가지고 있는가?
8. 우수한 임원진을 보유하고 있는가?
9. 경영 깊이가 있는가? (2인자, 3인자 품질)
10. 원가 분석과 회계 관리가 훌륭한가?

**경영진 품질**:
11. 산업 특유의 경쟁 우위가 있는가?
12. 장기적 이익을 중시하는가?
13. 성장 자금 조달을 위해 주식 희석이 필요한가?
14. 경영진이 좋을 때는 말하고 나쁠 때는 숨기는가?
15. 경영진의 정직성이 의심스럽지 않은가?

##### 5.3 집중 투자 + 장기 보유
```
포트폴리오: 5-10개 종목만
보유 기간: 10년+ (영원이 목표)

"주식 시장에서 돈을 버는 방법: 올바른 주식을 사서 오래 보유하는 것"
```

#### 대표 투자 성과
- Motorola (1955년 매수 → 40년 보유): 수백 배 수익

### 6. 캐시 우드 (Cathie Wood)

**회사**: ARK Invest (2014년 설립)

#### 핵심 철학
> "파괴적 혁신(Disruptive Innovation)에 투자하라."

#### 투자 전략

##### 6.1 5대 혁신 플랫폼
```
1. DNA 시퀀싱 (생명공학)
2. 로봇공학 (자동화)
3. 에너지 저장 (배터리, 태양광)
4. 인공지능 (AI)
5. 블록체인 (암호화폐, NFT)

투자 대상: 이 플랫폼들이 교차하는 기업
예: Tesla (AI + 배터리 + 로봇)
```

##### 6.2 라이트의 법칙 (Wright's Law)
```
"생산량이 2배 증가할 때마다 비용이 일정 비율로 감소"

예시:
- DNA 시퀀싱 비용: $10,000 (2007) → $600 (2023)
- 태양광 패널: 90% 가격 하락 (10년)

전략: 비용 하락 → 수요 폭발 → 투자!
```

##### 6.3 고집중 + 고변동성
```
포트폴리오: 30-50개 종목
상위 10개: 50%+ 비중

리스크: 변동성 매우 높음
- 2020: +150%
- 2021: -24%
- 2022: -67%

→ 장기 투자자만 적합
```

#### 대표 투자 성과
- ARK Innovation ETF (ARKK)
  - 2014-2020: 연평균 40%+ (S&P 500 대비 3배)
  - 2020-2023: 변동성 극심

## 📊 거시/매크로 투자 (Macro Investing)

### 7. 조지 소로스 (George Soros)

**회사**: Quantum Fund

#### 핵심 철학
> "시장은 항상 편향되어 있다. 중요한 것은 그 편향의 방향이다."

#### 투자 전략

##### 7.1 반사성 이론 (Reflexivity) ⭐⭐⭐⭐
```
시장 = 양방향 피드백 루프

일반적 믿음:
펀더멘털 → 가격

소로스의 반사성:
펀더멘털 ⇄ 가격 (양방향!)

예시:
1. 호황 → 주가 상승
2. 주가 상승 → 소비 증가 (자산 효과)
3. 소비 증가 → 실제 호황 강화
4. 반복 → 버블

→ 버블 정점에서 공매도!
```

##### 7.2 대형 매크로 베팅
```
소로스의 유명한 트레이드:

1. 영국 파운드화 공매도 (1992)
   - 수익: $1B+
   - 별명: "영란은행을 무너뜨린 남자"

2. 아시아 금융위기 (1997)
   - 태국 바트화 공매도

3. 2008 금융위기
   - 부동산 관련 파생상품 공매도

전략: 거시 경제 불균형 포착 → 레버리지 베팅
```

##### 7.3 틀렸을 때 빠르게 인정
```
"중요한 것은 맞는 횟수가 아니라, 맞았을 때 얼마나 버는가다."

손절 규칙:
- 틀렸다고 판단되면 즉시 청산
- 자존심 버리기
- 큰 손실 1번 > 작은 손실 10번

포지션 크기:
- 확신 높음: 큰 포지션
- 확신 낮음: 작은 포지션
```

#### 대표 투자 성과
- Quantum Fund (1973-2000): 연평균 30%+
- 영란은행 공매도 (1992): 하루 $1B 수익

### 8. 레이 달리오 (Ray Dalio)

**회사**: Bridgewater Associates (세계 최대 헤지펀드)

#### 핵심 철학
