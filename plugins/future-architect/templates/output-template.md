## 🧠 개념 관계도

```mermaid
graph TD
    A[전체 목표/비전] --> B[토픽 1]
    A --> C[토픽 2]
    A --> D[토픽 3]

    B --> B1[구체적 액션 1-1]
    B --> B2[구체적 액션 1-2]

    C --> C1[구체적 액션 2-1]
    C --> C2[구체적 액션 2-2]

    D --> D1[구체적 액션 3-1]
    D --> D2[구체적 액션 3-2]

    B -.enables.-> C
    D -.supports.-> C

    style A fill:#f9f,stroke:#333,stroke-width:4px
    style B fill:#bbf,stroke:#333,stroke-width:2px
    style C fill:#bbf,stroke:#333,stroke-width:2px
    style D fill:#bbf,stroke:#333,stroke-width:2px
```

**관계 설명**:
- 실선 화살표 (→): 상위-하위 관계
- 점선 화살표 (-.->): 토픽 간 관계
  - **enables**: A가 B를 가능하게 함
  - **depends**: B가 A에 의존함
  - **supports**: A가 B를 지원함
  - **conflicts**: A와 B가 자원을 경쟁함 (빨간 점선)
